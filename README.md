# pi-sms
Send SMS messages via Rpi using Node.js

# Requirements

    nodejs
    npm -- # Only if not previously installed, as nodemailer and mail-listener2 is installed via npm
    nodemailer
    mail-listener2

# Install pi-sms
    git clone https://github.com/TooManyEggrolls/pi-sms.git
    cd pi-sms
    ./install.sh
